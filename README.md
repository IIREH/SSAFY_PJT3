## 🚀 팀 소개



저희 팀원들은 1학기에 처음 웹을 시작한 인원들로 구성된 팀으로서,  

1학기 때의 부족함을 채우려 1학기 비전공자 반의 관통 프로젝트를 고도화하려 뭉친 팀입니다.



<SNS 웹 큐레이션>과 <추천 시스템 고도화>를 통해 영화 추천 서비스를 시작할 예정입니다.
